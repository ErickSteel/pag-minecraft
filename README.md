# pag-minecraft

### Requisitos:
```
Python 3.8.3
```
```
Django 3.1.1 - pip install Django
```
```
Graphene 2.13.0 - pip install graphene-django  
```
```
Django-filter 2.3.0 - pip install django-filter
```
```
Django GraphQL JWT 0.3.1 - pip install django-graphql-jwt
```
